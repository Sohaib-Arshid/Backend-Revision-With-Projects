# Full Stack Backend Practice Project 🚀

This project is a **Full Stack Web Application** built for learning and practicing modern backend and frontend development.

The goal of this project is to build a scalable system with **APIs, authentication, user management, and video handling**, while integrating a **React frontend** with a **Node.js / Express backend**.

---

## 📌 Project Goals

This project is being developed to practice and understand:

* REST API development
* Full stack architecture
* Authentication systems
* File & video handling
* Frontend and backend integration
* Real world backend concepts

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/Sohaib-Arshid/Backend-Revision-With-Projects
```

Move into the project folder:

```
cd Backend-Revision-With-Project
```

Install dependencies:

```
npm install
```

---

## ▶️ Running the Project

Start the backend server:

```
node index.js
```

Start the frontend:

```
npm run dev
```
